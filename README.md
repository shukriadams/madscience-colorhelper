# madscience-colorhelper

A NodeJS library of color functions.

## Use 

Add to package.json :

    {
        {
            "dependencies": {
                "madscience-colorhelper": "https://github.com/shukriadams/madscience-colorhelper.git#0.0.1"
        }
    }

Import

    const colorhelper = require('madscience-colorhelper').pager
